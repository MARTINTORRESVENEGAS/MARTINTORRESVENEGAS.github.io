Música:




Quieres insertar una pista de música o audio?

Para insertar pistas de audio existen 3 tipos de comandos

EL primero es el que se muestra a continuación

<audio controls>
<source src="horse.ogg" type="audio/ogg">
<source src="horse.mp3" type="audio/mpeg" >
<audio>

Quedando como resultado lo siguiente.

El uso de este comando nos permite tener los controles de el audio que vamos a insertar


El siguiente nos permite colocar un aduio con auto play es decir que se reprodusca de automaticamente al inicar la página


<audio controls autoplay>
<source src="horse.ogg" type="audio/ogg">
<source src="horse.mp3" type="audio/mpeg" >
<audio>

El siguiente nos permite hacer que el audio con autoplay no se escuche aunque este se esté reproduciendo


<audio controls autoplay muted>
<source src="horse.ogg" type="audio/ogg">
<source src="horse.mp3" type="audio/mpeg" >
<audio>

Videos:

Para insertar videos contamos con un comando similar al anterior el cual también cuenta con un control, un autoplay y un autoplay mute.

EL SIGUIENTE ES EL COMANDO QUE SE UTILIZA PRA COLOCAR UN VIDEO EN NUESTRA PÁGINA


<video width="320" height="240" controls>
<source src="movie.mp4" type="video/mp4">
<source src="movie.ogg" type="video/ogg">
</video>

ESTE FUNCIONA PARA PNER UN VÍDEO CON AUTOREPRODUCCIÓN


<video width="320" height="240" autoplay>
<source src="movie.mp4" type="video/mp4">
<source src="movie.ogg" type="video/ogg">
</video>

Y POR ULTIMO EL COMANDO ARA COLOCAR UN VIDEO CON AUTOREPRODUCCIÓN PERO SIN SONIDO

<video width="320" height="240" autoplay muted>
<source src="movie.mp4" type="video/mp4">
<source src="movie.ogg" type="video/ogg">
</video>
